  • Full Stack Jr

    #YoProgramo es la segunda etapa del plan Argentina Programa. En esta capacitación se aprenden distintas herramientas y tecnologías tanto para el Frontend como Backend. Para el Frontend se utiliza HTML, CSS y Javascript, dentro del framework de Angular., mientra que para la parte de Backend y Base de datos se utiliza Java y SQL (MySQL). Además, se aprende sobre el uso de Github para crear nuestros propios repositorios
